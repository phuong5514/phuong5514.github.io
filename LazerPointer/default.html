About This Page

Welcome to Lazer pointer's group page! This website was created as part of our web development coursework at the University of Science, VNU-HCM. As Computer Science students specializing in Software Engineering, We're passionate about building elegant and functional web applications.

This project showcases our understanding of modern web development practices, including responsive design, semantic HTML, CSS architecture using BEM methodology, and creating accessible user interfaces. The site demonstrates our ability to implement IBM's Carbon Design System principles while maintaining clean, maintainable code.

Through this project, We've applied key concepts such as mobile-first design, CSS Grid and Flexbox layouts, and progressive enhancement. We're continuously learning and improving our skills in web technologies, always striving to create better user experiences and write more efficient code.
